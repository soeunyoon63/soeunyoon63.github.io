ABOUT WORK

Hold, Please

Brand Identity, Motion

2019

Hold, Please is a campaign about the issue of a long hold for customer service call. I amplified this issue with a set of visual system which consists of a tagline, Hold please, a triadic color scheme, type marks, and audio in loop.

Software Used

Adobe Creative Suite

PROBLEM

We all have been in the situation where the waiting time to speak to a customer service representative is super long. There are so many problems with different types of urgency or difficulty because not every customer will be cooperative; thus, the audience should consider this inevitable problem not just as negatively but find ways to pass time a lot quicker.
